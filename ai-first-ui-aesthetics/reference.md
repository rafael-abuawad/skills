# A Color Bright: 14 visual identity trends (UI lens)

Compact definitions adapted from *Aesthetics of AI* (A Color Bright). Use as a pick-list, not a checklist—see [SKILL.md](SKILL.md).


| Trend                   | One-line lens for product UI                                                                                                          |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| Shades of off-white     | Quiet-luxury neutrals: warm off-whites, muted greys/beiges; low visual risk; trust and restraint over hype.                           |
| Organic gradients       | Soft, non-flat color fields; add grain/texture/subtle variation for an ownable look versus default AI gradients.                      |
| Digital impressionism   | Blurred, mood-led visuals; suggestive not literal; good for “thinking”/ambient states without over-claiming how AI works.             |
| Lomo imagery            | Imperfection as signal: contrast, wash, light leaks, happy accidents; exploratory, human, anti-corporate polish.                      |
| Contemporary realism    | Photoreal still-life or scenes: precision, craft, grounded capability; differentiates from vague decorative gradients.                |
| Sketch and scribble     | Whiteboard/notebook marks, uneven lines; traces of thought, exploration over certainty.                                               |
| Non-brand academia      | Lean type, muted palette, functional grid; cred through rigor, not gloss—research-notebook seriousness.                               |
| Technical illustrations | Diagram-adjacent decoration: schematics, retro-tech illustration; signals depth without pretending to be a real architecture diagram. |
| Quirky cuteness         | Mascots/faces/playful characters; approachability and humanization—use sparingly in dense pro UIs.                                    |
| Morphing objects        | Forms that drift, dissolve, reassemble; metaphor for emergence, learning, no final state—motion and loaders.                          |
| Futuristic surrealism   | Dreamlike worlds, impossible machines, retrofuturism; “gateway” framing—hero/marketing more than everyday chrome.                     |
| Outer space             | Cosmic scale, exploration metaphor; stars, galaxies, void—pairs with transformational or frontier positioning.                        |
| ASCII and pixels        | Early-computer nostalgia; friendly, technical, playable—dev tools, Easter eggs, micro-brand moments.                                  |
| Generative art          | Rule-based/system visuals: cellular patterns, emergent grids; “living system” metaphor for AI-native identity.                        |


**Source:** [A Color Bright — Insight](https://acolorbright.com/insight). Brand names in the original report are market context only; do not imitate a specific competitor wholesale.
